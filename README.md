# Login & Signup Demo using Node.js

A sample demo of login & signup

## Getting started

To get the project up and running, and view components in the browser, complete the following steps:

- Clone this repo
- To Run frontend : Open frontend folder and run any `.html` files on the browser
- Naviage to backend folder & `npm install` to install all required dependencies
- Install MongoDB Community Edition ([instructions](https://docs.mongodb.com/manual/installation/#tutorials)) and run it by executing mongod
- `npm start` to start the local server
- Open your browser and visit [http://localhost:3000](http://localhost:3000)
